# Foreign Homeland (2025)

**A Film by Armin Zarringhalam**

Foreign Homeland is a poignant cinematic journey tracking the first three weeks of an Iranian father and his three children after fleeing to Calgary, Canada. This single-page web experience serves as the official landing page for the film, featuring a detailed cast and crew gallery, world premiere highlights, and award recognitions.
